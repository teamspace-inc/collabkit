# CollabKitVueProvider

## Use CollabKit in a Vue.js application

`CollabKitVueProvider` is a Vue.js component that can be used instead of [CollabKitProvider](../core/collabkitprovider)
to configure CollabKit in a Vue.js application that uses [Veaury](https://github.com/devilwjp/veaury) to render our React components.

### Usage

Set up `CollabKitVueProvider` as the root component of your Vue.js app.
{renderCodeSnippet(CollabKitVueProviderUsage)}

Use Veaury and `CollabKitContextProvider` to render CollabKit React components elsewhere in the app.
{renderCodeSnippet(CollabKitContextProviderUsage)}
